SuiMail

CommandOSS Labs · 2025

A privacy-first decentralized email protocol on the Sui blockchain — architected and built full-stack, from Move smart contracts to Rust backend services and a Next.js frontend.

Spam-gating engine. Per-mailbox GatingConfig Move objects + a Rust engine classifying mail into drop / pass-free / toll-payment flows — a cryptoeconomic anti-spam layer.
End-to-end encryption. Seal IBE so only intended recipients decrypt, with no key-exchange overhead.
Censorship-resistant storage. A Walrus pipeline for tamper-proof on-chain message storage with off-chain availability.
Human-readable identity. SuiNS addressing (user@suins.sui) maps names to on-chain mailboxes instead of wallet addresses.
Web2 → Web3 auth. zkLogin lets users sign in via Google/Apple OAuth without managing private keys.
TEE custodial wallets. Oyster CVM key protection using Shamir Secret Sharing + AES-256-GCM for sender key management.
